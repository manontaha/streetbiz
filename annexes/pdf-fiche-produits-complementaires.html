Comment remplir ce tableau
  • Produit : le nom du produit complémentaire (ex : Coca 33cl, chips nature, brownie maison…)
  • Cond. : la quantité vendue dans le lot fournisseur (ex : pack de 24, lot de 100…)
  • Tarif HT : le prix hors taxes du lot fournisseur
  • Tarif TTC : HT × 1,055 si alimentaire · HT × 1,20 si non alimentaire
  • Coût unitaire : Tarif TTC ÷ Conditionnement = le coût d'une unité
  • Prix min. conseillé : Coût unitaire × 3 — c'est le Coefficient StreetBiz™
Exemples de produits complémentaires
Boissons : Coca-Cola 33cl, eau minérale, jus de fruits, limonade maison — 3 maximum conseillés
Chips / snacks : sachet de chips, biscuits salés, crackers — 2 maximum conseillés
Desserts : brownie maison, cookie, madeleine — 1 maximum conseillé
