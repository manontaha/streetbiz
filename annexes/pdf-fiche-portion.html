🥗 Ingrédients
Comment remplir ce tableau
  • Ingrédient : le nom du produit utilisé dans cette portion (ex : blanc de poulet, tomate, pita…)
  • Qté : la quantité que tu utilises pour UNE portion (ex : 150, 2, 30…)
  • Unité : l'unité de mesure (g, ml, unité, cl…)
  • A/M : A = Acheté tel quel chez le fournisseur · M = Maison (tu le prépares toi-même, voir ta fiche-recette)
  • Fournisseur / Recette : le nom du fournisseur si A, ou le nom de ta recette si M
  • Prix TTC / Coût total : le prix TTC du conditionnement fournisseur si A · le coût total de ta recette si M
  • Cond. / Rendement : la quantité totale vendue par le fournisseur si A · le rendement de ta recette si M
  • Coût réel : le coût de cet ingrédient pour UNE portion. Formule : Quantité demandée par la portion × Prix TTC du fournisseur ÷ Conditionnement fournisseur

📦 Emballages & accessoires
Comment remplir ce tableau
  • Emballage / accessoire : ce que tu utilises pour emballer ou servir (ex : boîte kraft, barquette, serviette, sachet papier…)
  • Qté : le nombre utilisé pour UNE portion (généralement 1)
  • Fournisseur : le nom du fournisseur (ex : Raja, Retif…)
  • Prix TTC : le prix TTC du lot acheté (ex : 12,00 €)
  • Conditionnement : le nombre d'unités dans le lot (ex : 50 boîtes, 100 serviettes…)
  • Coût réel : le coût pour UNE portion. Formule : Quantité demandée par la portion × Prix TTC du fournisseur ÷ Conditionnement fournisseur
