Comment remplir cette fiche
  • Produit : le nom de l'ingrédient ou de l'emballage (ex : blanc de poulet, boîte kraft…)
  • Type : coche Ingrédient ou Emballage — la formule TVA à utiliser s'affiche à côté
  • Fournisseur : le nom du fournisseur (ex : Metro, Grand Frais, Raja…)
  • Tarif HT : le prix hors taxes affiché par le fournisseur pour ce lot
  • Conditionnement : la quantité totale dans le lot (ex : 1 kg, lot de 50, 5 L…)
  • Tarif TTC : applique la formule selon le type. Entoure le meilleur fournisseur.
